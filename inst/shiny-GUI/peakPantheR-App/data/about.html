peakPantheR

Peak Picking and ANnoTation of High resolution Experiments in R

peakPantheR is designed to detect and integrate multiple compounds in multiple files in parallel and store results in a single object.

To extract compounds intensity from data files, the following information is required:

  • Path(s) to netCDF / mzML MS file(s)
  • An expected region of interest (RT / m/z window) for each compound

The following steps take place during parallel compound annotation:

  • Data Import: Load a list of regions of interest and files to process.
  • Detection and integration: Detect features in each region of interest, integrate and calculate peak statistics
  • Diagnostic: Update regions of interest, plot the resulting fits and return the corresponding statistics
  • Export: Save the integration values and fit statistics, plot the extracted ion chromatograms

Abbreviations:

  • EIC: extration ion chromatogram
  • RT: retention time
  • m/z: mass over charge ratio
  • ROI: region of interest, region to search for a targeted compound
  • uROI: updated region of interest, refined search region
  • FIR: Fallback integration region, region to integrate if not valid peak can be found

1 - Import Data

Inputs:

  • Path(s) to netCDF / mzML MS file(s)
  • An expected region of interest (RT / m/z window) for each compound

The output object will be initialised with the targeted ROI and file paths.

2 - Detection and Integration

Once the object is initialised, the data can be processed.

For each file, features in each region of interest are detected, the highest intensity feature is integrated and peak statistics determined.

Results and EICs for each ROI are stored in the output object.

3 - Diagnostic

Based on the results obtained, updated regions of interest (uROI) and fallback integration regions (FIR, if no feature is detected (median RT / m/z start and end of found features)) can be determined.

The resulting fit can be visually inspected and diagnostic plots (all EICs, peak apex RT / m/z and peak width evolution) generated.

4 - Export

The integration values (row: file, col: compound), fit statistics, EIC plots and resulting object can be saved.
